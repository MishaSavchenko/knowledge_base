* Linux File System:
	* Three Layers
		* Logical File System -> user application and the file system 
		* Virtual File System -> standard interface for compatibility 
		* Physical File System -> management of physical memory blocks
	* ext4
		* popular, stable, balanced, easy to use, can be optimized
	* xfs
		* good for large file storage, harder to setup and configure, good for media
	* btrfs
		* built in redundancy for really important files
